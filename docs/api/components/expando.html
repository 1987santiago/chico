Cómo se usa

El Expando es un componente que se expande y colapsa para mostrar información relacionada a un titular determinado. Por defecto se encuentra colapsado y al clickear en el titular se expande, mostrando la información que estaba oculta.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").expando();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	onShow: function() {},		   // Callback executed on expando show
	onHide: function() {},		   // Callback executed on expando hide
	open: boolean					// Expand configuration. Default "false"	
}

Ejemplo:

$(".foo").expando({
	onHide: function() {
		// something
	}
});

Objeto público

Cada vez que se instancia un componente expando, retorna un objeto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").expando();

// returns
foo = {
	uid: 121,						//  Global number instance
	element: HTMLElement,			//  DOM Interface
	type: "expando",			     //  Component Type
	show(),						  //  Show expando
	hide() 						  //  Hide expando	
}

	content: "Callbacks al exapandir y colapsar",
	onShow: function() {
		alert("Callback onShow");
	},
	onHide: function() {
		alert("Callback onHide");
	}
});
