Cómo se usa

El Tab Navigator se crea a partir de agregarle funcionalidad a un conjunto de anclas.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").tabNavigator();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	selected: number,				   // Opened tab by default 
	onSelect: function() {},    		// Callback executed after show modal
	onContentLoad: function() {},	   // Callback executed on success of content loaded via ajax
	onContentError: function() {} 	  // Callback executed on fail of content loaded via ajax
}

Ejemplo:

$(".foo").tabNavigator({
	selected: 2
});

Objeto público

Cada vez que se instancia un componente Tab Navigator, retorna un objeto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").tabNavigator();

// It returns
foo = {
	uid: 121,								 //  Global number instance
	element: HTMLElement,					 //  DOM Interface
	type: "tabNavigator",			     	//  Component Type
	children: Array						   //  Instances of components controlled by Tab Navigator
	select(): function(tab:Number) {},		//  Select the tab that should be open
	getSelected(): function() {}	  		//  Return the opened tab	
}

Si queremos ejecutar un callback cuando se selecione un tab en especial, debemos evaluar que tab fue seleccionado. Para ello, utilizamos la propiedad getSelected():

$("#example3").tabNavigator({
	onSelect: function(){
		if( this.getSelected() == 2 ){  
			alert("Se seleccionó el tab número: 3");
		};	
	}
});
