Para que el Carousel pueda realizar todos los cálculos internos, es necesario especificar vía CSS, el alto y ancho de los elementos de la lista:

.demoCarousel li {
	width: 110px;
	height: 110px;
}
 
$(".demoCarousel").carousel();

Cómo se usa

El Carousel se crea a partir de agregarle funcionalidad a una lista de objetos, ya sean imágenes, texto, o ambos.

Para crear un Carousel, es necesario iniciarlo a traves de un contenedor, en el cual puede haber todo tipo de elementos, y una lista con la clase CSS "carousel". Es decir, que el Carousel puede convivir dentro de una caja junto con contenido de cualquier tipo.

El ancho y alto de los elementos <li> del Carousel deben ser especificados obligatoriamente por CSS.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").carousel();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	onMove: function() {}			  // Callback executed after carousel movement (It is triggered when use next(), prev() or moveTo() methods)
}

Ejemplo:

$(".foo").carousel({
	onMove: function() {
		// something
	}
});

Objeto público

Cada vez que se instancia un componente Carousel retorna un objeto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").carousel();

// It returns
foo = {
	uid: 121,                        //  Global number instance
	element: HTMLElement,	        //  DOM Interface
	type: "carousel",                //  Component Type
	getSteps(),					  //  Returns items amount of each page
	getPage(),					   //  Returns actual page
	moveTo(page:Number),             //  Select a page
	next(),						  //  Move to next page
	prev(),						  //  Move to previous page
}

Ejemplos

1. Mover hacia una página específica

Mediante el objeto público del Carousel, se puede seleccionar una determinada página usando el método moveTo(), al cual se le especifica el número de página al cual moverse.

var carouselMove = $(".example1").carousel();

$(".btn_example1").click(function(){
	carouselMove.moveTo(2);
});

2. Mover hacia a las páginas anterior y siguiente

Mediante el objeto público del Carousel, se puede navegar hacia adelante y hacia atras entre las páginas usando los métodos next() y prev() respectivamente.

var carouselNav = $(".example2").carousel();

$(".btn_prev_example2").click(function(){
	carouselNav.prev();
});

$(".btn_next_example2").click(function(){
	carouselNav.next();
});

3. Callback al moverse el Carousel

El callback onMove() se ejecuta luego de moverse el Carousel, ya sea cliqueando los botones Anterior y Siguiente, o utilizando los métodos públicos moveTo(), next() o prev().

$(".example3").carousel({
	onMove: function() {
		alert("Callback onMove");
	}
});
