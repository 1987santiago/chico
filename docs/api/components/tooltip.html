Cómo se usa

El Tooltip es un componente emergente que puede iniciarse con cualquier elemento HTML que tenga el atributo title. Se dispara con el evento mouseover del elemento, o bien, utilizando su método público show().

Este componente pertenece a la familia Floats (al igual que Modal window y Contextual Layer).

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").tooltip();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	onShow: function() {},    		  // Callback executed after show tooltip
	onHide: function() {},			  // Callback executed before hide tooltip
	offset: string,					 // Positioner configuration. Default "0 10"
	points: string,					 // Positioner configuration. Default "lt lb"
	hold: boolean					   // Positioner configuration. Default "false"
}

Ejemplo:

$(".foo").tooltip({
	onShow: function() {
		// Do something
	},
	onHide: function() {
		// Do something
	}
});

Objeto público

Cada vez que se instancia un componente Tooltip, retorna un objecto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").tooltip();

// It returns
foo = {
	uid: 121,                         //  Global number instance
	element: HTMLElement,        	 //  DOM Interface
	type: "tooltip",                  //  Component Type
	content: string,				  //  Get content
	position(),					   //  Get, set and refresh position
	show(),						   //  Show layer
	hide()						    //  Hide layer
}

Posicionamiento

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

foo.position();					   //  Get position

2. Actualizar la posición del elemento.

foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

foo.position({ 					   //  Set new position
	context: $(element),
	offset: string,
	points: string,
	hold: boolean
});

Ejemplos

1. Callbacks al mostrarse y ocultarse

El callback onShow() se ejecuta luego de mostrarse el Tooltip, mientras que el callback onHide() se ejecuta antes de ocultarse el componente:

$(".example1").tooltip({
	onShow: function() {
		alert("Callback onShow");
	},
	onHide: function() {
		alert("Callback onHide");
	}
});
