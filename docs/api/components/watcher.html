Cómo se usa

El objeto Watcher define una nueva familia de componentes que se encargan de validar campos de formularios. Recibe como argumento un objeto de propiedades para su funcionamiento, mensajes o para los callbacks.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").component();

Se asocia a uno o más controles de formularios y se encarga agrupar las validaciones por cada uno. Cada instancia de Watcher posee una referencia directa con el elemento <form> padre. Cada elemento de formulario debería estar relacionado con una instancia del componente Form de la familia Controllers. De esta manera logramos una referencia circular desde un elemento de formulario, la instancia de Watcher, el elemento <form> y su instancia de Form.

Existen varias interfaces que permiten la utilización de los Watchers. Se dividen en 4 grupos: String, Number, Required y Custom. Cada uno de ellos define internamente un Watcher con configuraciones especiales de validación.

Lista de interfaces disponibles:

  • required()
  • string()
  • email()
  • url()
  • maxLength()
  • minLength()
  • Number()
  • max()
  • min()
  • price()
