$(".demoDropdown").dropdown();

Cómo se usa

El objeto Dropdown es un componente que agrupa una lista de links, los cuales disparan diferentes acciones. Puede recibir como argumento un objeto de propiedades para los callbacks.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").dropdown();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	onShow: function() {},		   // Callback executed on dropdown show
	onHide: function() {},		   // Callback executed on dropdown hide
	offset: string,				  // Positioner configuration. Default "0 -1"
	points: string,				  // Positioner configuration. Default "lt lb"
	hold: boolean					// Positioner configuration. Default "false"	
}

Ejemplo:

$(".foo").dropdown({
	onHide: function() {
		// something
	}
});

Objeto público

Cada vez que se instancia un componente expando, retorna un objeto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").dropdown();

// returns
foo = {
	uid: 121,						//  Global number instance
	element: HTMLElement,			//  DOM Interface
	type: "dropdown",				//  Component Type
	position(),					  //  Get, set and refresh position
	show(),						  //  Show dropdown
	hide(),						  //  Hide dropdown	
}

Posicionamiento

El método público position() puede ser utilizado para:

1. Obtener la configuración actual de posicionamiento.

foo.position();					   //  Get position

2. Actualizar la posición del elemento.

foo.position("refresh");			  //  Refresh position

3. Setear una nueva configuración para el posicionamiento del elemento.

foo.position({ 					   //  Set new position
	context: $(element),
	offset: string,
	points: string,
	hold: boolean
});
