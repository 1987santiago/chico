$(".demoViewer").viewer();

Cómo se usa

El Viewer se genera a partir de extender la funcionalidad de un listado de imágenes de tamaño medio, las cuales poseen un link a una imagen de mayor tamaño.

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").viewer();

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	onMove: function() {}			  // Callback executed after Viewer movement (It is triggered when use next(), prev() or moveTo() methods)
}

Ejemplo:

$(".foo").viewer({
	onMove: function() {
		// Do something
	}
});

Objeto público

Cada vez que se instancia un componente Viewer retorna un objecto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").viewer();

// It returns
foo = {
    uid: 121,                        //  Global number instance
    element: HTMLElement,	        //  DOM Interface
    type: "viewer",                  //  Component Type
    children: array,                 //  Instances of components controlled by Viewer
    getSelected(),				   //  Returns actual item selected
    moveTo(page),                    //  Select an item
    next(),						  //  Move to next page
    prev(),						  //  Move to previous page
}

Ejemplos

1. Mover hacia un item específico

Mediante el objeto público del Viewer, se puede seleccionar un determinado item usando el método moveTo(), al cual se le especifica el número de item al cual moverse.

var viewerMove = $(".example1").viewer();

$(".btn_item2_example1").click(function(){
	viewerMove.moveTo(2);
});

$(".btn_item6_example1").click(function(){
	viewerMove.moveTo(6);
});

2. Mover hacia a los items anterior y siguiente

Mediante el objeto público del Viewer, se puede navegar hacia adelante y hacia atras entre los items usando los métodos next() y prev() respectivamente.

var viewerNav = $(".example2").viewer();

$(".btn_prev_example2").click(function(){
	viewerNav.prev();
});

$(".btn_next_example2").click(function(){
	viewerNav.next();
});
