Get

Cómo se usa

El objeto Get es una utilidad que permite la carga asincrónica de contenidos o componentes permitiendo una carga a demanda. Esta funcionalidad habilita al implementador configurar componentes ante la acción del usuario. Este componente permite aplicar fácilmente la estrategia de mejora progresiva.

Ejemplo de configuración:

{
    method: cons,               // "content" or "component"
    component: string,          // name of the component
    [script]: url,              // need to specify a script ?
    [style]: url,               // need to specify some styles ?
    [callback]: function(){}    // when the thing is loaded
}

Internamente cada componente puede solicitar contenidos por AJAX:

ui.get({
    method:"content",           // Go and get content
    conf: { ... }               // Component internal configuration
});

También podemos solicitar la carga asincrónica de un componente abstracto:

ui.get({
    "method": "component",              // Go and get a component
    "component": "problemSolver",       // The name
    "script": "src/js/component.js",    // Location of the script
    "styles": "src/css/styles.css",     // Some decorative details
    "callback": function() {            // What to do next
        // do something awesome here
        // like if you were a ninja 
    }
        });

O simplemente cargar algunos estilos a demanda:

ui.get({
    "method": "component",              // Go and get a component
    "component": "somenewstyles",       // The name
    "styles": "src/css/styles.css",     // Some decorative details
    }
});
