Factory

Cómo se usa

El objeto Factory se encarga de configurar componentes. Está definido en el scope del Core y se integra con el componente Get para obtener recursos si es necesario.

Ejemplo de configuración:

{
    component: string,              // Name the component
    [script]: url,                  // Where is located ?
    [style]: url,                   // Need some styles ?
    [callback]: function () {}      // Do something when loads
}

Si se trata un componente que existe en el namespace del framework, solo define el nombre:

ui.factory({
      component: "chat",
      callback: function(){
          // chat loaded    
      }
});

Si se trata de algún plug-in externo, que está alojado en otro lado:

ui.factory({
      component: "chat",
      script: "http://domain.com/chat.js",
      callback: function(){
          // chat loaded    
      }
});

Además el Factory automáticamente va a crear un shortcut en el namespace para instanciar el componente cargado:

$("#chat_window").chat();
