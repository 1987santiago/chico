Core

Cómo se usa

El objeto Core define un namespace de nombre ui como módulo principal de Chico-UI. Este patrón permite alojar todos los componentes en un contexto para evitar colisiones de nombre con otras librerías. Además provee algunas propiedades globales que los componentes pueden acceder.

Algunas propiedades:

ui.version      // return the version
ui.components   // return a string with all components
ui.internals    // return a string with all internal functions

Cada instancia de componentes visuales se guardan en un mapa interno que agrupa cada una por tipo de componente:

ui.instances = {
    layer: [ ... ],
    modal: [ ... ]    
};

Esto nos permite recorrer multiples instancias de distintos tipos y ejecutar alguna acción en un grupo de componentes. Por ejemplo podría cerrar todos los componentes Layer en una simple iteración:

$(ui.instances.layer).each(function(i,e){   // Iterate all layer instances
    e.hide();                               // Hide each one of them
});

Además el Core posee un grupo de propiedades y métodos útiles para cualquier componente:

ui.utils = {
    window: object Window,          // Shortcut to the window object
    document: object HTMLDocument   // Shortcut to the document object
    body: object HTMLBodyElement    // Reference to the body tag 
    html: object HTMLHtmlElement    // Reference to the html tag 
    index: int,                     // show the last assigned index for ui components
    zIndex: int                     // show the last assigned z-index for floats       
}
