Cómo se usa

El Positioner es un componente que tiene como función posicionar elementos con respecto a un contexto.

El contexto puede ser cualquier elemento del DOM o el area visible de la ventana del navegador (o viewport).

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

Existen dos maneras de iniciar el componente; como objeto global, a partir de un elemento, o desde una instancia de componente:

ui.positioner();				//  Global object
component.position();		   //  Through component public instance

Points

Como parte de la configuración, el Positioner puede recibir una referencia a los puntos en común por los cuales el contexto y el elemento deben posicionarse.

Para especificar este comportamiento se definieron una serie de puntos con la siguiente estructura:

"lt lb"

Las primeras dos letras representan al punto de referencia del elemento a posicionar, mientras que las segundas dos letras representan al punto de referencia del contexto con respecto al cual se posicionará el elemento. Ambos grupos están separados por un espacio.

La primera letra de cada una de estas dos agrupaciones, representa al eje horizontal (left, center o right), y la segunda letra representa al eje vertical (top, middle o bottom).

Las posibles combinaciones de posicionamiento son las siguientes:

Puntos Descripción Ubicación del elemento con respecto al contexto
cm cm Element center-middle axis equal to Context center-middle axis Center
lt lb Element left-top axis equal to Context left-bottom axis Bottom
lb lt Element left-bottom axis equal to Context left-top axis Top
rt rb Element right-top axis equal to Context right-bottom axis Bottom
rb rt Element right-bottom axis equal to Context right-top axis Top
lt rt Element left-top axis equal to Context right-top axis Right
lm lm Element left-middle axis equal to Context left-middle axis Right
lt lt Element left-top axis equal to Context left-top axis Right

Para tener en cuenta: En próximas versiones de Chico, se completaran todas las combinaciones de posicionamiento.

Configuración

La mayoría de los componentes pueden recibir parámetros de configuración, que modifican el comportamiento por defecto.

Objeto de configuración:

{
	element: $(element),		//  Element to be positioned
	context: $(element),		//  Context from which the element is positioned. Default: viewport
	offset: string,			 //  A string with X and Y coordinates (in pixels) separated by space (i.e.: "10 0"). Default "0 0"
	points: string,			 //  A string with element and context shared axis, separated by space (i.e.: "lt lb"). Default "cm cm" (centered)
	hold: boolean			   //  Hold element's position. Default "false"
}

Ejemplo como objeto global:

ui.positioner({
	element: $(".foo"),
	context: $(".context"),
	offset: "10 0",
	points: "cm cm"
});

Ejemplo de configuración desde una instancia de componente:

component.position({
	context: $(".context"),
	offset: "10 0",
	points: "cm cm"
});
