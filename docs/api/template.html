Cómo se usa

Realizando una selección con jQuery podemos iniciar el componente de la siguiente forma:

$(".foo").component();

Objeto de configuración:

{
	name: value	// Comment
}

Ejemplo:

$(".foo").component({
	name: value
});

Otra forma de iniciarlo es utilizando la siguiente sintaxis:

$(".foo").component(string or number or function);

Cada vez que se instancia un componente ***component*** retorna un objecto público de control que nos brinda información sobre la instancia.

var foo = $(".foo").component();

// It returns
foo = {
	uid: 121,                         //  Global number instance
	element: HTMLInputElement,        //  Input DOM Interface
	type: "modal",                    //  Component Type
}
