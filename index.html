Instalación

Para la instalación es necesario tener en cuenta las precarga de algunas dependencias de la librería.

  • Dependencias
  • Crossbrowsing
  • Extensiones

Para utilizar Chico UI son necesarias las siguientes dependencias que deben ir linkeadas en el código HTML. No es necesario descargarlas.

  • Esta es la versión del base.css que reemplaza al base que se este utilizando y no se encuentre en la carpeta chico:

    http://10.100.34.210:8080/content/chico/0.x.x/src/css/base.css

  • En caso de utilizar formularios, deben agregar los siguientes estilos:

    http://10.100.34.210:8080/content/chico/0.x.x/src/css/forms.css

  • jQuery:

    http://10.100.34.210:8080/content/chico/0.x.x/src/js/jquery.js

  • Este javascript es el único que se necesita para inicializar los componentes del SDK:

    http://10.100.34.210:8080/content/chico/0.x.x/src/js/mlui.js

Para que los componentes del SDK soporten crossbrowsing es necesario reemplazar el <body> por estas lineas para activar los modificadores de estilos para IE. Estas moficaciones permiten ajustar automáticamente las diferencias más básicas entre los dispositivos.


<!--[if lt IE 7 ]>
<body class="ie6">
<script src="http://10.100.34.210:8080/content/chico/0.x.x/src/js/dd_belatedpng.js"></script>
<script> DD_belatedPNG.fix('img, .ico'); //fix any <img> or .ico background-images </script>
<![endif]-->
<!--[if IE 7 ]> <body class="ie7"> <![endif]-->
<!--[if IE 8 ]> <body class="ie8"> <![endif]-->
<!--[if IE 9 ]> <body class="ie9"> <![endif]-->
<!--[if (gt IE 9)|!(IE)]> <!--> <body> <!--<![endif]-->

Extender funcionalidades de Chico UI es sencillo y rápido. Todos los componentes visuales de Chico UI se definene en el namespace ui.* y heredan de un constructor genérico que posee la funcionalidad básica para integrarse con el SDK. Para extender funcionalidad del constructor genérico o de cualquier otra familia de componentes utilizamos un patrón de OOP muy ágil y sencillo presentado por Douglas Crockford llamado PowerConstructor.

Ejemplo componente Foobar:

			ui.Foobar = function(foo){	// Foobar Constructor
				var data = foo;	// Datos privados
				var that = ui.PowerConstructor(); // Defino la herencia
					that.member = {}; // Miembros públicos
					that.toString = function() { return "Foo Object"; }
					return that; // Devuelvo el objeto creado
			}

Para configurar el componente es necesario:

  1. Crear un archivo con el nombre del componente (ej: "Foobar.js") con el código y colocarlo en la ruta /src/js/.
  2. Agregar las hojas de estilo en la ruta /src/css/ con el mismo nombre de archivo.
  3. Agregar el nombre del componente a la lista de componentes disponibles en el núcleo del SDK.
			ui.components += "Foobar";

Ahora el componente esta listo para usarse:

			$("div.foo").Foobar({data:"foo"});

Eso es todo ;)
